Web Development Insights

Learning web design is a journey full of challenges and fun. During this process, I deeply realized that daring to imagine and trying bravely are the keys to improving skills and perfecting works.

Inspiration is the Starting Point of Design

Every web design starts with a new idea. I often ask myself: "What if I design the navigation bar as floating?" "Can the page turn like a book?" These bold ideas became my motivation to explore web design. Even if I don't know how to achieve it at first, I won't limit my imagination.

The Creation Process from Scratch

Once I have an idea, I immediately start trying. The code I write at the beginning may be rough, and the page may not be satisfactory, but every attempt brings new gains. When a floating navigation bar successfully appears on the screen, or an interactive animation runs smoothly, the sense of accomplishment motivates me to challenge new goals.

Gradual Improvement: From Trying to Refinement

Web design is a continuous optimization process. I find problems in each attempt and keep adjusting. For example, a dynamic effect may look stiff at first, but by modifying parameters or adding transition effects, it can become smoother. Through such small improvements, my web pages gradually become more refined from simple to complex, from rough to exquisite.

AI Assistance: Becoming My Design Partner

Throughout the process, AI played an important role. Whenever I encounter difficulties, AI can always provide timely help, from basic code to solving complex logic problems. Its guidance saved me a lot of detours. Moreover, AI inspired me with many new ideas, such as how to achieve subtle gradient effects with CSS or add interactive functions with JavaScript. This companion-like support keeps my enthusiasm for design high.

ChatGPT Gemini Copilot GitHub Copilot

Conclusion

By daring to think and try, my web pages have grown from simple layouts to works full of personality and functionality. This experience made me realize that the charm of design lies in continuous attempts and progress, and the companionship of AI adds more possibilities to this journey. In the future, I look forward to creating more wonderful web pages with this exploratory spirit and AI support.

Technical Summary

Holy Grail Layout

When the page is wide enough, the homepage is a three-column layout, with the main content in the middle and sidebars on both sides, implemented using flex layout.

Responsive Layout

When the page width is small, the various parts of the webpage are arranged in columns, implemented using media queries. This better adapts to different screen sizes and provides a better experience on mobile devices.

Dynamic Background

Using the animation property to achieve a constantly changing gradient background, beautiful and magnificent.

Various CSS Styles

Including fonts, colors, borders, shadows, margins, rounded corners, transparency, etc. As you can see, these styles make my webpage more beautiful and refined.

Giscus Discussion Area

Implemented using Giscus discussion area + GitHub discussions, enabling interaction with users on a static page.

Simple Games with H5

Originating from the mini-games I previously published on GitHub. It includes both front-end implementation and back-end calculations. Here, its style has been modified to fit the page style.
